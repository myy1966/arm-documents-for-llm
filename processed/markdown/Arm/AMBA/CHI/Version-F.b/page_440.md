## B13.10 Protocol flit fields

- B13.10.2 Target Identifier, TgtID

- B13.10.4 Home Node Identifier, HomeNID

- B13.10.14 Transaction Identifier, TxnID

- B13.10.17 Data Buffer Identifier, DBID
